## Install

```bash
npm install --save brsp-scripting-chat
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'brsp-scripting-chat'
import 'brsp-scripting-chat/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
